Ingeniero de Software Senior

Tech Solutions Inc. · Remoto

Descripción del puesto

Tech Solutions Inc. está buscando un Ingeniero de Software Senior altamente motivado y experimentado para unirse a nuestro equipo remoto. El candidato ideal tendrá un sólido conocimiento de las últimas tecnologías, experiencia en el desarrollo de aplicaciones web y la capacidad de liderar equipos de software escalables y de alto rendimiento. Como Ingeniero de Software Senior, será responsable de crear soluciones efectivas y de calidad, colaborar con equipos multifuncionales y supervisar funciones importantes para entregar productos de software excepcionales a nuestros clientes.

Responsabilidades

  • Diseñar, desarrollar y mantener aplicaciones web utilizando tecnologías modernas.
  • Colaborar con equipos de producto y diseño para definir y refinar especificaciones técnicas.
  • Escribir código limpio, eficiente y bien documentado.
  • Realizar revisiones de código y proporcionar retroalimentación constructiva a los miembros del equipo.

Requisitos

  • Licenciatura en informática o campo relacionado.
  • Mínimo 5 años de experiencia en desarrollo de software.
  • Experiencia con frameworks en JavaScript (por ejemplo, React, Angular, Vue.js).
  • Familiaridad con metodologías ágiles y herramientas de control de versiones (por ejemplo, Git).

Acerca de la empresa

Tech Solutions Inc. se dedica a ofrecer tecnología innovadora que está transformando la manera en que las empresas operan. Nuestro equipo está compuesto por profesionales apasionados que valoran la colaboración, la creatividad y la excelencia técnica. Nos enfocamos en entregar productos de software de alta calidad que resuelvan problemas complejos, impulsen el crecimiento empresarial y generen valor para nuestros clientes.
